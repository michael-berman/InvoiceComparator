# InvoiceComparator
Application to parse invoices and compare different items bought from suppliers

## Technologies
* Django 3.0.7
* Python 3.8.3
* JQuery 3.1.3
* AWS S3

## Features
* Upload and parse invoices using OCR
* Upload items with manual form
* Comparison table
* AWS storage for invoices
* Deployed to Heroku with CDCI
